# vim

## 配置

vim ~/.vimrc

```bash
set nu # 设置行号
set tabstop=4 # tab宽度4个空格
set expandtab # tab转换为4个空格
set autoindent # 自动缩进
set encoding=utf-8 #设置默认编码
```
