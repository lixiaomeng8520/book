# 安装

### 系统需求

* 包含url重写的web服务器
* PHP 5.5或更高

### 如何安装Slim

我们建议你通过[Composer](https://getcomposer.org/)来安装。进入到你项目的根目录，然后执行下面的命令。这个命令下载Slim框架和第三方依赖到项目的 `vendor/` 目录。

```bash
composer require slim/slim "^3.0"
```

将composer自动加载包含到你的PHP脚本里，下面就可以使用Slim了。

```php
<?php
require 'vendor/autoload.php';
```
